# After Sign-Off

## Overview
A nostalgic **retro TV simulator** that recreates the classic television experience of the 1980s-90s era. This interactive project features a fully functional TV set interface with channel switching, visual effects, and authentic analog TV aesthetics.

## Features
- **Authentic TV Cabinet Design**: Realistic wood-grain finish with beveled edges and control knobs
- **Retro Screen Effects**: 
  - CRT scanlines for authentic cathode-ray tube appearance
  - Phosphor glow effects mimicking old television phosphorescence
  - Screen glass reflection overlay
- **Interactive Controls**:
  - Channel switching via physical knob controls
  - Space bar and arrow key support
  - ON AIR indicator LED
  - Channel strip buttons
- **Canvas-Based Graphics**: Smooth animated visuals on a canvas element

## Technical Stack
- **HTML5**: Semantic markup with canvas element
- **CSS3**: Advanced styling with animations and visual effects
- **Vanilla JavaScript**: Interactive controls and canvas animations

## How to Use
1. Open `index.html` in a modern web browser
2. Click the knobs or channel buttons to switch channels
3. Use space bar or arrow keys as keyboard shortcuts
4. Watch the authentic retro TV effects unfold on the canvas

## Browser Compatibility
Works best on modern browsers with HTML5 Canvas and CSS3 support (Chrome, Firefox, Safari, Edge)

## Files
- `index.html` - Main markup with TV structure
- `style.css` - Styling and visual effects
- `script.js` - Interactive functionality and canvas rendering
