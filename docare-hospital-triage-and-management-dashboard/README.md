# DoCare - Hospital Triage and Management Dashboard

## Overview
A comprehensive **healthcare management system** designed for hospital triage and patient flow optimization. This interactive dashboard enables medical staff to efficiently manage patient queues, priority levels, and notifications with dual-view interfaces for doctors and patients.

## Features
- **Dual-View Interface**:
  - Doctor View: Complete patient management and triage system
  - Patient Screen: Display queue and current service status
- **Patient Management**:
  - Priority-based queue system (Critical, Urgent, Standard)
  - Patient ID and status tracking
  - Automatic queue progression
  - Patient history logging
- **Interactive Controls**:
  - Hamburger navigation menu
  - Context menu for quick actions
  - View toggle between doctor and patient displays
  - Sound announcements for new patients
- **Notification System**:
  - Real-time patient announcements
  - Bell notifications
  - Audio alerts
  - Queue status updates
- **Responsive Design**:
  - Works on desktop and mobile displays
  - Touch-friendly interface
  - Adaptive layouts

## Technical Stack
- **HTML5**: Semantic markup with comprehensive structure
- **CSS3**: Professional styling with animations and responsive design
- **Vanilla JavaScript**: State management, event handling, and notification logic
- **Font Awesome**: Icon library for UI elements

## How to Use
1. Open `index.html` in a web browser
2. Switch between Doctor View and Patient Screen using the toggle buttons
3. Use context menu (right-click) for patient management options
4. Click the loudspeaker icon to announce patients
5. Monitor the notification system for queue updates

## Features Breakdown

### Doctor Dashboard
- View all patients in queue
- Sort by priority level
- Access patient information
- Call next patient
- Update patient status
- Print/export reports

### Patient Screen
- Display current serving number
- Show estimated wait time
- Display department information
- Simple, clear queue status

Works on all modern browsers (Chrome, Firefox, Safari, Edge) with responsive support
