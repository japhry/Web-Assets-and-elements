### For Full Collection

```bash

cd Web-Assets-and-elements
```

2. Choose a project and explore
3. Customize as needed for your use case

### Quick Start Examples

**Run a project locally:**
```bash
# Navigate to project
cd product-cards

# Open in default browser (Windows)
start index.html

# Or on macOS
open index.html

# Or on Linux
xdg-open index.html
```

**Compile SASS/SCSS projects:**
```bash
sass style.scss style.css
```

**Compile TypeScript projects:**
```bash
tsc script.ts
```

**Compile Pug templates:**
```bash
pug index.pug
```
