# Neumorphic Form

## Overview
A **neumorphism design pattern implementation** featuring a modern form interface. This project showcases the soft UI/neumorphic trend with subtle shadows, minimalist aesthetics, and refined form elements that blend seamlessly with the background.

## Features
- **Neumorphic Design**:
  - Soft shadows for depth
  - Inset and outset effects
  - Minimalist color palette
  - Monochromatic styling
- **Form Elements**:
  - Text input fields
  - Password fields
  - Checkbox controls
  - Radio buttons
  - Submit button
- **Interactive States**:
  - Focus effects
  - Hover animations
  - Active states
  - Validation feedback
- **Accessibility**:
  - Semantic HTML
  - Proper form labels
  - Keyboard navigation
  - Screen reader support

## Technical Stack
- **HTML5**: Semantic form structure
- **CSS3**: 
  - Box-shadow for neumorphic effect
  - CSS custom properties (variables)
  - Flexbox layout
  - Smooth transitions
- **Minimal JavaScript**: Form validation and interactions

## How to Use
1. Open `index.html` in a web browser
2. Fill in the form fields
3. Observe the neumorphic shadow effects
4. Click to focus on inputs
5. Submit the form to see validation

## Neumorphism Characteristics
- **Soft Shadows**: Subtle, non-aggressive shadows
- **Monochromatic Color**: Single color with variations
- **Blended Design**: Elements blend into background
- **Minimal Contrast**: Low contrast for soft appearance
- **Depth**: Use of inset/outset shadows for layering

## Customization
- Adjust `--shadow-color` variable for different neumorphic effects
- Modify `--bg-color` for different color schemes
- Change border-radius for different roundness levels
- Customize shadow blur and spread for different depths

## Browser Compatibility
Works on all modern browsers:
- Chrome 49+
- Firefox 31+
- Safari 12.1+
- Edge 15+
