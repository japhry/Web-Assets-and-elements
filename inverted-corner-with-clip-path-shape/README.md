# Inverted Corner with Clip-Path Shape

## Overview
An advanced **CSS clipping demonstration** featuring inverted corners and custom shape masks. This project showcases the power of CSS `clip-path` to create unique geometric designs and modern UI elements with sharp, inverted corners.

## Features
- **CSS Clip-Path Masking**:
  - Custom shape creation without SVG
  - Inverted corner effects
  - Complex polygon shapes
  - Dynamic shape transformations
- **Visual Design**:
  - Modern geometric aesthetics
  - Sharp, angular corners
  - Creative frame designs
  - Smooth animations
- **Interactive Elements**:
  - Hover effects
  - Click animations
  - Shape transitions
  - Dynamic content display

## Technical Stack
- **HTML5**: Semantic markup with content containers
- **CSS3**: 
  - clip-path property for shape masking
  - CSS animations and transitions
  - Flexbox layout
  - Advanced selectors
- **Vanilla JavaScript**: Interactive functionality

## How to Use
1. Open `index.html` in a modern web browser
2. Observe the inverted corner effects on various elements
3. Hover over elements to see animations
4. Click to trigger interactive effects
5. Study the CSS to understand clip-path implementation

## CSS Clip-Path Examples
- Inverted corners on cards
- Triangular shapes
- Polygon designs
- Custom button shapes
- Frame borders with cutouts

## Customization
- Modify the `clip-path` values in `style.css` for different shapes
- Adjust animation timings for different effects
- Change colors and backgrounds for your design
- Scale the shapes using CSS transforms

## Browser Compatibility
Works on all modern browsers:
- Chrome 25+
- Firefox 49+
- Safari 13.1+
- Edge 79+

## Files
- `index.html` - Elements with clip-path shapes
- `style.css` - Clip-path definitions and animations
