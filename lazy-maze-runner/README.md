# Lazy Maze Runner

## Overview
An engaging **interactive maze game** with lazy evaluation and performance optimization. This project showcases game development techniques with efficient maze generation, smooth animations, and a challenge-based gameplay system.

## Features
- **Maze Generation**:
  - Procedurally generated mazes
  - Multiple difficulty levels
  - Optimized pathfinding algorithms
- **Gameplay Mechanics**:
  - Player character navigation
  - Obstacle avoidance
  - Goal-reaching objectives
  - Score and time tracking
- **Performance Optimization**:
  - Lazy rendering techniques
  - Efficient collision detection
  - Optimized animations
- **Visual Effects**:
  - Smooth character movement
  - Animated obstacles
  - Visual feedback for actions
  - UI overlays for score and time

## Technical Stack
- **HTML5**: Canvas for game rendering
- **CSS3**: UI styling and animations
- **Vanilla JavaScript**: 
  - Game loop implementation
  - Maze algorithms
  - Physics and collision detection
  - Lazy evaluation patterns

## How to Use
1. Open `index.html` in a web browser
2. Use Arrow Keys or WASD to move the player
3. Navigate through the maze to reach the goal
4. Avoid obstacles and time limits
5. Complete levels to progress

## Game Controls
- **Arrow Keys** or **WASD**: Move character
- **Space**: Perform action (jump/dash)
- **R**: Restart current level
- **N**: Next level
- **P**: Pause game

## Difficulty Levels
- **Easy**: Simple maze, more time
- **Medium**: Complex paths, standard time
- **Hard**: Dense obstacles, time pressure
- **Expert**: Maximum challenge with all features

## Browser Compatibility
Requires modern browsers with HTML5 Canvas and ES6 support

## Files
- `index.html` - Game container and UI
- `style.css` - Game styling and UI layout
- `script.js` - Game logic, maze generation, and physics
