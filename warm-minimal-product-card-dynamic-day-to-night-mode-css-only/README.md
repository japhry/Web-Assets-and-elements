# Warm Minimal Product Card - Dynamic Day to Night Mode CSS Only

## Overview
An elegant **product card component** featuring a dynamic day-to-night theme toggle implemented purely with CSS. This project showcases modern e-commerce design with automatic dark mode support, warm color palettes, and minimal yet sophisticated styling.

## Features
- **Day/Night Theme Toggle**:
  - CSS-only theme switching
  - No JavaScript required
  - Smooth transitions between themes
  - Respects system preferences
- **Product Card Design**:
  - Product image display
  - Title and description
  - Price information
  - Rating display
  - Stock status
- **Warm Color Palette**:
  - Inviting, warm tones
  - Daylight colors
  - Nighttime colors
  - High contrast readability
- **Minimal Aesthetic**:
  - Clean design
  - Ample whitespace
  - Elegant typography
  - Subtle shadows
- **Interactive Elements**:
  - Hover effects
  - Smooth transitions
  - Call-to-action buttons
  - Interactive rating

## Technical Stack
- **HTML5**: Semantic product structure
- **SCSS**: 
  - CSS custom properties (variables)
  - Media queries for system preferences
  - Nested selectors
  - Mixins and functions
- **Pure CSS**: No JavaScript for theme switching

## How to Use
1. Open `index.html` in a web browser
2. Observe the current theme (based on system preference)
3. View product information
4. Hover over elements to see interactions
5. Click to interact with buttons
6. Check system dark mode to see theme change

## Theme Features
- **Automatic Detection**: Respects `prefers-color-scheme`
- **Smooth Transitions**: Color changes animate smoothly
- **Day Theme**: Bright, warm, inviting colors
- **Night Theme**: Dark, comfortable colors for low-light viewing
- **All Elements**: Complete theme coverage

## CSS Variables Used
- `--bg-color`: Background color
- `--text-color`: Text color
- `--accent-color`: Accent/primary color
- `--shadow-color`: Shadow colors
- `--border-color`: Border colors

## Customization
