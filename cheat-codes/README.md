# Cheat Codes

## Overview
An interactive **secret key sequence detector** that implements classic video game cheat code functionality. This project demonstrates event-driven programming and keyboard input handling with a retro gaming aesthetic.

## Features
- **Konami Code Detection**: Recognizes famous cheat code sequences from gaming history
- **Visual Feedback**:
  - Real-time key press display
  - Success animation when complete code is entered
  - Notification system for code activation
- **Multiple Cheat Sequences**: Support for various gaming cheat patterns
- **Smooth Animations**: Stylish UI transitions and effects

## Technical Stack
- **Slim Template Language**: Concise markup syntax for HTML generation
- **Vanilla JavaScript**: Keyboard event handling and sequence detection
- **Stylus CSS**: Dynamic stylesheet language with variables and mixins

## How to Use
1. Open `index.html` in a web browser
2. Press the secret key sequence (e.g., Up, Up, Down, Down, Left, Right, Left, Right, B, A)
3. Watch for visual feedback when the correct sequence is entered
4. Experiment with different cheat codes

## Cheat Code Patterns
The project detects classic gaming sequences:
- Konami Code
- Street Fighter codes
- Other retro gaming patterns

## Browser Compatibility
Works on all modern browsers with keyboard event support

## Files
- `index.slim` - Template markup (Slim language)
- `style.stylus` - Advanced CSS with Stylus features
- `script.js` - Keyboard detection and sequence matching logic
