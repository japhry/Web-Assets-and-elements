# Frosted Saturated Borders

## Overview
A stunning **modern UI component** featuring decorative borders with frosted glass and vibrant saturated colors. This design showcases contemporary web design trends with blurred glassmorphism effects combined with bold color palettes.

## Features
- **Glassmorphism Effect**:
  - Frosted background blur using CSS backdrop-filter
  - Semi-transparent backgrounds
  - Elegant transparency layering
- **Saturated Color Palette**:
  - Vibrant, eye-catching colors
  - Modern gradient combinations
  - High contrast styling
- **Border Animations**:
  - Dynamic border transitions
  - Hover effects with smooth animations
  - Gradient border effects
- **Responsive Layout**:
  - Flexible container sizing
  - Mobile-friendly design
  - Adaptable to different screen sizes

## Technical Stack
- **HTML5**: Semantic markup structure
- **CSS3**: 
  - Backdrop filters for glass effect
  - CSS Gradients
  - Flexbox layout
  - Advanced animations
- **Vanilla JavaScript**: Interactive animations and event handling

## How to Use
1. Open `index.html` in a modern web browser
2. Hover over elements to see animations
3. Click elements to trigger interactions
4. Observe the frosted glass effects
5. Customize colors in `style.css` as needed

## Customization
- Modify the color variables in `style.css` for different color schemes
- Adjust border radius and blur intensity for different effects
- Change animation timing in the CSS for faster/slower transitions

## Browser Compatibility
Requires modern browsers with CSS backdrop-filter support:
- Chrome 76+
- Firefox 103+
- Safari 9+
- Edge 79+

## Files
- `index.html` - Component markup and structure
- `style.css` - Glassmorphism effects and animations
- `script.js` - Interactive functionality and event listeners
