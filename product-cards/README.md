# Product Cards

## Overview
A beautiful **e-commerce product card component** showcasing modern design patterns for online retail. This project demonstrates responsive product displays with hover effects, product information, and call-to-action elements perfect for shopping websites and product catalogs.

## Features
- **Product Card Design**:
  - Product image display
  - Title and description
  - Price and rating
  - Stock availability indicator
- **Interactive Effects**:
  - Hover animations
  - Image zoom effects
  - Smooth transitions
  - Shadow depth effects
- **Action Elements**:
  - Add to cart button
  - Wishlist toggle
  - Quick view option
  - Rating display
- **Responsive Layout**:
  - Grid-based product layout
  - Mobile-optimized cards
  - Flexible sizing
  - Touch-friendly buttons

## Technical Stack
- **HTML5**: Semantic product structure
- **CSS3**: 
  - CSS Grid for layouts
  - Flexbox for component layout
  - Transform animations
  - Smooth transitions
- **Vanilla JavaScript**: Interactive features and cart logic

## How to Use
1. Open `index.html` in a web browser
2. Browse the product cards
3. Hover over cards to see animations
4. Click "Add to Cart" to add products
5. Toggle wishlist hearts
6. View product details

## Product Card Elements
- **Image Section**: Product photo with zoom on hover
- **Product Info**: Name, description, and details
- **Rating**: Star rating and review count
- **Price**: Regular and sale prices
- **Actions**: Add to cart and wishlist buttons
- **Status**: Availability and stock indicators

## Customization
- Modify card width and spacing for grid layout
- Adjust animation timings for different effects
- Change colors to match brand guidelines
- Add new product cards by duplicating HTML structure
- Customize button text and actions

## Browser Compatibility
Works on all modern browsers:
- Chrome 50+
- Firefox 50+
- Safari 10+
- Edge 15+

## Use Cases
- E-commerce websites
- Product catalogs
- Shopping applications
- Marketplace platforms
- Store display systems
