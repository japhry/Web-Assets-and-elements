# Unblocked Games

## Overview
A **game portal collection** featuring unblocked, browser-based games. This project showcases a catalog of playable games in a clean, accessible interface that works in restricted network environments where traditional gaming sites are blocked.

## Features
- **Game Collection**:
  - Multiple playable games
  - Variety of game genres
  - Easy game selection
  - Quick launch interface
- **Game Library**:
  - Game descriptions
  - Thumbnail previews
  - Difficulty indicators
  - Category tags
- **Game Management**:
  - Game search functionality
  - Category filtering
  - Favorites system
  - Recently played tracking
- **User Interface**:
  - Clean, intuitive layout
  - Responsive design
  - Easy navigation
  - Mobile-optimized

## Technical Stack
- **HTML5**: Semantic game catalog structure
- **CSS3**: 
  - Grid-based game gallery
  - Card designs
  - Responsive layouts
  - Smooth transitions
- **Vanilla JavaScript**: 
  - Game selection logic
  - Search and filter functionality
  - Game launch controls

## How to Use
1. Open `index.html` in a web browser
2. Browse the available games
3. Click on a game to view details
4. Click "Play" to launch the game
5. Use search to find specific games
6. Filter by category

## Game Categories
