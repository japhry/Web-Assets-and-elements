# Pure CSS 3D Animated Carousel - Minimal Code Heavily Commented

## Overview
A cutting-edge **pure CSS 3D carousel** implemented with minimal, heavily-commented code for educational purposes. This project demonstrates advanced CSS 3D transforms and animations without relying on JavaScript, perfect for learning CSS capabilities and 3D perspective techniques.

## Features
- **Pure CSS 3D Transforms**:
  - 3D perspective and rotation
  - transform-style: preserve-3d
  - translateZ and rotateY effects
  - Realistic 3D depth
- **Automatic Carousel**:
  - Self-rotating carousel
  - Smooth 3D transitions
  - Continuous animation loop
  - Auto-play functionality
- **Interactive Controls**:
  - Pause on hover
  - Manual navigation (CSS only)
  - Smooth animation timing
  - Elegant easing functions
- **Responsive Design**:
  - Mobile-friendly display
  - Adaptive sizing
  - Touch-friendly interface
  - Flexible layouts

## Technical Stack
- **Pug Template Language**: Clean, indentation-based markup
- **CSS3**: 
  - 3D transforms
  - Perspective properties
  - CSS animations (no JavaScript)
  - Media queries for responsiveness
- **Minimal JavaScript**: Optional enhancements only

## Key Features
- **No JavaScript Required**: Pure CSS animations
- **Educational Comments**: Extensive code comments explaining every part
- **Clean Code**: Minimal HTML structure
- **High Performance**: Optimized CSS animations
- **Cross-browser Compatible**: Works on modern browsers

## How to Use
1. Open `index.html` (or compile `.pug` file) in a web browser
2. Watch the carousel auto-rotate
3. Hover over carousel to pause
4. Study the comments in `style.scss` to understand CSS 3D
5. Modify values to see different effects

## Understanding CSS 3D
This carousel teaches:
- **Perspective**: How to set up 3D space
- **Transform Origin**: Positioning rotation points
- **TranslateZ**: Creating depth along Z-axis
- **RotateY**: Creating rotation around Y-axis
- **Animation Timing**: Smooth, continuous rotations
- **Preservation of 3D**: Maintaining 3D transformations on children

## Customization
- Adjust `perspective` value for different depth
- Modify animation duration for faster/slower carousel
- Change `rotateY` angle for different rotation amounts
- Adjust card count and spacing
- Customize colors and styling
