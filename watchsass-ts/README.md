# Watch SASS/TS

## Overview
A sophisticated **watch/clock design showcase** built with SASS and TypeScript. This project demonstrates the combination of preprocessor technologies to create a beautiful timepiece visualization with interactive features and modern styling techniques.

## Features
- **Watch Display**:
  - Analog watch face
  - Hour, minute, and second hands
  - Numerical or marker indices
  - Rotating second hand
- **Real-Time Updates**:
  - Live time display
  - Smooth hand animations
  - Current time synchronization
  - Timezone support
- **Interactive Elements**:
  - Adjustable time
  - Speed controls
  - Style customization
  - Theme options
- **Design Elements**:
  - Elegant watch casing
  - Premium styling
  - Smooth animations
  - Professional typography

## Technical Stack
- **TypeScript**: 
  - Type-safe time calculations
  - Object-oriented design
  - Improved code maintainability
  - Better IDE support
- **SASS**: 
  - Nested selectors
  - Mixins and functions
  - CSS variables
  - Advanced styling techniques
- **HTML5**: Semantic structure

## How to Use
1. Compile TypeScript: `tsc script.ts`
2. Compile SASS: `sass style.sass style.css`
3. Open `index.html` in a web browser
4. Watch the live time display
5. Interact with controls to adjust display
6. Customize styling and theme

## TypeScript Features
- **Time Calculation**: Accurate angle calculations for hands
- **Class-Based**: Object-oriented clock implementation
- **Type Safety**: Interfaces and type definitions
- **Event Handling**: Robust event management

## SASS Features
- **Variables**: Reusable color and size values
- **Mixins**: Reusable style patterns
- **Functions**: Dynamic style calculations
- **Nesting**: Organized stylesheet structure
- **Extends**: Style inheritance and reuse

## Compilation Commands
```bash
# TypeScript compilation
tsc script.ts

# SASS compilation
sass style.sass style.css

# Watch mode (auto-compile on changes)
tsc --watch script.ts
sass --watch style.sass:style.css
```
