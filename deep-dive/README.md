# Deep Dive

## Overview
An immersive **underwater exploration experience** with dynamic animations and visual effects. This project showcases advanced CSS animations combined with JavaScript interactivity to create an engaging deep-sea diving simulator.

## Features
- **Underwater Environment**:
  - Animated water effects and wave simulations
  - Particle system for bubbles and floating debris
  - Depth-based color and lighting effects
  - Marine life animations
- **Interactive Elements**:
  - Player navigation controls
  - Depth meter display
  - Oxygen level monitoring
  - Sound effects for immersion
- **Dynamic Animations**:
  - Smooth character movement
  - Creature animations
  - Environmental parallax scrolling
  - Light ray effects

## Technical Stack
- **HTML5**: Semantic markup with canvas for animations
- **CSS3**: Complex animations, transforms, and effects
- **Vanilla JavaScript**: Game logic and event handling

## How to Use
1. Open `index.html` in a modern web browser
2. Use keyboard controls (Arrow Keys or WASD) to navigate
3. Explore the underwater environment
4. Interact with creatures and objects
5. Monitor your depth and oxygen levels

## Controls
- **Arrow Keys** or **WASD**: Move around
- **Space**: Perform action
- **Escape**: Return to surface

## Browser Compatibility
Requires modern browsers with HTML5 Canvas, CSS3 animations, and ES6 JavaScript support

## Files
- `index.html` - Game structure and canvas setup
- `style.css` - Environmental styling and animations
- `script.js` - Game logic, physics, and interactivity
